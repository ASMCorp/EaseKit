# EaseKit Documentation

This directory contains the EaseKit documentation site, built with Jekyll and GitHub Pages.

## Building Locally

```bash
cd docs
bundle install
bundle exec jekyll serve
```

## Structure

- `index.html` - Landing page
- `getting-started.md` - Installation guide
- `api-reference.md` - Complete API documentation
- `examples.md` - Code examples and comparisons
- `_config.yml` - Jekyll configuration
- `assets/` - CSS, images, and static files
