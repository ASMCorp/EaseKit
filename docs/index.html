{% comment %} JSON-LD Structured Data {% endcomment %} {% comment %} FAQ Structured Data {% endcomment %} ## Why EaseKit? EaseKit is a **free Swift Package Manager library** designed to eliminate boilerplate code in iOS development. Whether you're building with UIKit or SwiftUI, EaseKit provides the helpers you need to write cleaner, more maintainable Swift code.

🎨 Hex Color Extensions

Create UIColor and SwiftUI Color instances from hex strings. Supports 3, 6, and 8 character formats with optional alpha. Perfect for design system colors and dynamic theming.

let brandColor = UIColor(hexString: "#007AFF")
let semiBlue = Color(hex: "007AFF", alpha: 0.5)
let randomColor = UIColor.random

📐 Auto Layout Helpers

Simplify UIView Auto Layout with intuitive extension methods. Anchor views to edges, fill superviews, center views, and set dimensions with a single method call instead of verbose constraint code.

myView.anchorView(
    top: parent.topAnchor,
    leading: parent.leadingAnchor,
    paddingTop: 20, paddingLeft: 16
)
logoView.center(in: parentView)
backgroundView.fillSuperView()

🖼️ View to Image

Convert any UIView to a UIImage with a single method call. Useful for generating thumbnails, sharing view content, or creating snapshots for testing.

let cardView = CardView()
// ... configure the view ...
let image = cardView.asImage()

📦 Resource URL Helper

Create URLs for bundle resources with a clean, type-safe initializer. Works with both main bundle and Swift Package resources.

if let url = URL(localResourceName: "config", extensionName: "json") {
    // Load the resource
}
## Quick Installation ### Swift Package Manager (Xcode) 1. Open your Xcode project 2. Select **File → Add Package Dependencies...** 3. Enter: `https://github.com/ASMCorp/EaseKit.git` 4. Click **Add Package** ### Package.swift ```swift dependencies: [ .package(url: "https://github.com/ASMCorp/EaseKit.git", from: "1.0.0") ] ``` Then add `EaseKit` to your target's dependencies: ```swift .target( name: "YourApp", dependencies: ["EaseKit"] ) ``` ## Quick Start ```swift import EaseKit // Hex colors in UIKit let primaryColor = UIColor(hexString: "#007AFF") let backgroundColor = UIColor(hexString: "#F2F2F7") // Hex colors in SwiftUI struct ContentView: View { var body: some View { Text("Hello, EaseKit!") .foregroundColor(Color(hex: "#007AFF")) .background(Color(hex: "#F2F2F7", alpha: 0.5)) } } // Auto Layout with UIView extensions let container = UIView() let label = UILabel() label.text = "Hello, World!" container.addSubview(label) label.anchorView( top: container.topAnchor, leading: container.leadingAnchor, trailing: container.trailingAnchor, paddingTop: 16, paddingLeft: 20, paddingRight: 20 ) label.setDimension(height: 44) // Center a view let logo = UIImageView(image: UIImage(named: "logo")) container.addSubview(logo) logo.center(in: container) logo.setDimension(width: 120, height: 120) ``` ## Requirements | Requirement | Version | |-------------|---------| | iOS | 12.0+ | | Swift | 6.2+ | | Xcode | 14.0+ | ## Open Source & Free EaseKit is released under the [Apache 2.0 License](https://github.com/ASMCorp/EaseKit/blob/main/LICENSE).
